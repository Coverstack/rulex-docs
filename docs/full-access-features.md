# Full Access Features

The demo at [demo.rulex.coverstack.in](https://demo.rulex.coverstack.in/) is free to
use. It covers the core workflow: uploading Data Sources, defining Schemas, configuring
JSON API Endpoints, and exporting them for self-hosting.

The following features require full hosted access:

## Web Form Mode

Full access Endpoints can be exposed as browser-accessible HTML forms. Submitting the
form runs your Excel model and displays the results on the same page.

[Learn more about Web Form Mode](configuring/web-form-mode.md)

## Web Form Themes

Web forms can be styled using custom themes built with
[daisyUI](https://daisyui.com/theme-generator/). Themes support separate light and dark
variants.

[Learn more about Web Form Themes](configuring/web-form-themes.md)

## Web Form Credentials

Web forms are protected with a username and password. Unauthenticated visitors see
a login page before they can access the form.

[Learn more about Web Form Credentials](configuring/web-form-credentials.md)

## Endpoint Tests

You can test an Endpoint against multiple inputs at once by uploading an Excel file.
RuleX runs each row through the model and returns a results file.

[Learn more about Endpoint Tests](endpoint-tests.md)
