# Supported Formulas

RuleX supports 74 Excel formulas. All supported formulas behave identically to
Excel 2021.

If a formula in your model is not on this list, the Data Source build will fail with
an error. Check the build status and error log in RuleX Admin if your Source does not
compile.

| List of formulas |  |  |  |  |
|---|---|---|---|---|
| ABS | ANCHORARRAY | AND | AVERAGE | CEILING |
| CONCATENATE | COUNTA | COUNTBLANK | COUNTIF | COUNTIFS |
| DATE | DATEDIF | DAY | EXP | FALSE |
| FILTER | FLOOR | HLOOKUP | IF | IFERROR |
| IFNA | IFS | INDEX | IRR | ISBLANK |
| ISERR | ISERROR | ISNA | ISNONTEXT | ISREF |
| ISTEXT | JOIN | LEFT | LOWER | MATCH |
| MAX | MID | MIN | MONTH | NOT |
| NOW | OFFSET | OR | POWER | PRODUCT |
| PROPER | RAND | RANDBETWEEN | REPT | RIGHT |
| ROUND | ROUNDDOWN | ROUNDUP | SEARCH | SEQUENCE |
| SIGN | SQRT | SQRTPI | SUBSTITUTE | SUM |
| SUMIF | SUMSQ | SUMX2MY2 | SUMX2PY2 | SUMXMY2 |
| TEXTJOIN | TODAY | TRIM | TRUE | UPPER |
| VLOOKUP | XLOOKUP | XMATCH | YEAR | |

Need a formula that is not listed? Contact
[support@coverstack.in](mailto:support@coverstack.in).
